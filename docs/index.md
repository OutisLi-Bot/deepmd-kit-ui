# DeePMD Studio

DeePMD Studio is the standalone Tauri desktop application, Ratatui terminal
workbench, and JSONL agent interface for the DeePMD-kit Python toolchain.

The bundled runtime, Stable/Beta/Custom source channels, TUI commands, local
development flow, and release packaging are documented in the repository
[`README.md`](../README.md).
